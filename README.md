##  Problem Statement
With the rise of high-frequency digital transactions, manual fraud review is unscalable. This project implements a **Machine Learning solution** deployed as a **Microservice**, allowing for real-time fraud detection.

##  How to Run Locally

### Option 1: Streamlit Frontend (Recommended)
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the Streamlit app:
   ```bash
   streamlit run app.py
   ```

3. Open your browser at `http://localhost:8501`
